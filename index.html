F-droid használata


1. Távolítsd el az összes Novy Software alkalmazást. 2. Telepítsd az f-droid-ot (https://f-droid.org/F-Droid.apk) 3. Add hozzá a Novy Software repo-t: Kattints ide és nyiss meg f-droid-dal 4. Keresd meg, töltsd le és élvezd a Novy Software appokat!
